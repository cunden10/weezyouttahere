üîá Microphone Issues

Problem: "Microphone permission denied" or no audio detected

  1. Check browser permissions: Click the üîí icon in the address bar
  2. Ensure microphone access is allowed for this site
  3. Try refreshing the page and granting permission again
  4. Check system microphone settings and ensure it's not muted

üîå Connection Problems

Problem: "Connection failed" or frequent disconnections

Solution:

  1. Check your internet connection stability
  2. Disable VPN or proxy temporarily
  3. Try refreshing the page
  4. Contact support if issues persist

üìù Transcription Accuracy

Problem: Poor transcription quality or errors

Solution:

  1. Speak clearly and at a moderate pace
  2. Reduce background noise when possible
  3. Select the correct language in settings
  4. Use a high-quality microphone if available
  5. Ensure stable internet connection

‚ö° Performance Issues
